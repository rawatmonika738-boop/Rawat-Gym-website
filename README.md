## 📖 Overview

The Rawat Gym Website is a simple fitness website built using HTML and
CSS showcasing gym information, services, machines, membership plans and
a contact form.

## 📂 Repository Structure

``` text
Rawat-Gym/
├── index.html
├── gym.webp
├── README.md
```

## ✨ Key Features

-   About Us
-   Services
-   Imported Machines
-   Membership Plans
-   Information Form
-   Responsive Layout

## 🚀 Getting Started

Clone the repository and open `index.html` in your browser.
